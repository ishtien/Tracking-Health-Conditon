README

1.先在登入頁面打上0000
2.登入後可選擇兩種模式：健身模式、瘦身模式
3.點一下page會顯示當周的體重變化
    *顯示照片：當周身材變化照片
    *顯示詳細資料：日期
    *左右按鈕：選擇上周或下周資料
4.選單可切換頁面
5.今日菜單：可隨機生成今日的菜單，詳細資訊顯示菜單的熱量與注意事項
  健身菜單：
    *左邊格子為每天目標運動量，可勾選今天完成的運動
    *按察看進度後中間格子會顯示今日進度以及還必須完成的部分
    *右邊格子可打入想要的新進度，更改後左邊格子的選項就會改變
6.輸入當日體重：打入體重並選擇今日照片，按更新按鈕新增今日資料
7.主題：可改變背景圖片、字的背景顏色、音樂
8.目標：可更改目標的體重，並提供身高體重bmi等資訊
  *明星照片可更改增加動機
